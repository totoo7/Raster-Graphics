 A C++ raster graphics processor supporting common image formats and operations, designed with modularity and extensibility in mind.

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{index_autotoc_md2}
This project provides a suite of classes and commands to load, manipulate, and save raster images in the following formats\+:


\begin{DoxyItemize}
\item {\bfseries{\doxylink{classPPM}{PPM}}} (Portable Pixmap)
\item {\bfseries{\doxylink{classPGM}{PGM}}} (Portable Graymap)
\item {\bfseries{\doxylink{classPBM}{PBM}}} (Portable Bitmap)
\end{DoxyItemize}

It features core image processing operations such as\+:


\begin{DoxyItemize}
\item Grayscale conversion
\item Monochrome (black \& white) conversion
\item Negative (color inversion)
\item Rotation (clockwise and counterclockwise)
\item Flipping (horizontal and vertical)
\end{DoxyItemize}

The design is object-\/oriented, employing\+:


\begin{DoxyItemize}
\item Abstract base classes for images and commands
\item \doxylink{classCommand}{Command} pattern for applying and undoing operations
\item \doxylink{classSession}{Session} management for handling multiple image editing sessions
\end{DoxyItemize}

\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md4}

\begin{DoxyItemize}
\item Load and save \doxylink{classPPM}{PPM}, \doxylink{classPGM}{PGM}, and \doxylink{classPBM}{PBM} images
\item Perform common image transformations and filters
\item Undo/redo support for image commands
\item \doxylink{classCommand}{Command} factory for extensible command creation
\item \doxylink{classSession}{Session} management to isolate multiple image workflows
\item Doxygen-\/generated documentation including class diagrams and inheritance graphs
\end{DoxyItemize}
